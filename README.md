# Wisdom Wedding Songs

## 1. 프로젝트 소개

Wisdom Wedding Songs는 결혼식 축가 예약 및 관리 서비스입니다.

개인 고객 또는 웨딩샵이 결혼식 축가를 신청할 수 있으며,
관리자는 축가 신청 내역을 확인하고 상담 및 예약 상태를 관리할 수 있습니다.

기존에는 결혼식 축가를 구하기 위해 지인 소개 또는 웨딩샵 문의에 의존하는 경우가 많았지만,
Wisdom Wedding Songs는 축가 신청 과정을 보다 편리하게 관리할 수 있도록 제작한 프로젝트입니다.

이 프로젝트는 Spring Boot 기반의 백엔드 프로젝트로,
축가 신청 등록, 조회, 수정, 삭제(CRUD) 기능과 예약 상태 관리 기능을 중심으로 구현하였습니다.

## 5. 기능 설계

### 1차 개발 범위

이번 프로젝트의 1차 목표는 축가 신청을 등록하고 관리할 수 있는 기본 백엔드 API를 구현하는 것입니다.

### 주요 기능

| 기능 | 설명 |
|---|---|
| 축가 신청 등록 | 개인 고객 또는 웨딩샵이 결혼식 축가를 신청합니다. |
| 축가 신청 목록 조회 | 등록된 축가 신청 목록을 조회합니다. |
| 축가 신청 상세 조회 | 선택한 축가 신청의 상세 정보를 조회합니다. |
| 축가 신청 수정 | 신청자가 비밀번호를 입력하여 신청 내용을 수정합니다. |
| 축가 신청 삭제 | 신청자가 비밀번호를 입력하여 신청을 삭제합니다. |
| 신청 상태 변경 | 관리자가 신청 상태를 변경합니다. |

### 신청 상태

| 상태값 | 의미 |
|---|---|
| REQUESTED | 신청 접수 |
| CONSULTING | 상담 중 |
| CONFIRMED | 예약 확정 |
| COMPLETED | 완료 |
| CANCELED | 취소 |

### 신청자 유형

| 유형 | 의미 |
|---|---|
| CUSTOMER | 개인 고객 |
| WEDDING_SHOP | 웨딩샵 |

## 6. 결제 정책

Wisdom Wedding Song의 축가 예약 비용은 총 30만원이며,
예약금과 잔금을 분리하여 결제를 진행합니다.

| 구분 | 금액   | 결제 방식 | 결제 시점 |
|---|------|---|---|
| 예약금 | 20만원 | 카드 결제 | 예약 신청 시 |
| 잔금 | 20만원 | 계좌이체 | 결혼식 3일 전 |

예약금 결제가 완료되어야 예약이 확정되며,
잔금 미입금 시 예약 상태가 변경될 수 있습니다.
